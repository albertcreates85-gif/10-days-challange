# Python Mastery: 3-Project Portfolio
By **Mayank Gariya**

This repository contains a collection of three Python projects built to demonstrate my transition from fundamental syntax to professional-grade application development. Each project focuses on a different core pillar of the Python ecosystem: API integration, Web Scraping, and Data Analysis.

---

## 🚀 Project 1: Real-Time Weather & Wardrobe Advisor
**Focus:** *APIs, JSON Parsing, and OOP*

A professional-grade Weather Advisor that fetches live data from the OpenWeatherMap API and uses logic-driven recommendations to suggest a wardrobe based on current temperature and conditions.

### Key Features:
- **Object-Oriented Design:** Encapsulated logic within a `WeatherAdviser` class.
- **Dynamic API Handling:** Fetches real-time data for any city worldwide.
- **Error Management:** Robust handling for invalid API keys or city names.

---

## 🕵️ Project 2: Automated Deal Hunter
**Focus:** *Web Scraping & Data Cleaning*

An automated web scraper built using `BeautifulSoup4` and `Requests` to monitor e-commerce sites for price drops.

### Key Features:
- **DOM Navigation:** Precisely targets HTML tags and classes to extract product titles, prices, and stock status.
- **User-Agent Spoofing:** Mimics real browser headers to bypass basic bot-detection.

## 📊 Project 3: Smart Data Report Summarizer
**Focus:** *Data Science, Pandas, & Visualization*

A data analysis tool that takes "messy" financial CSV data and turns it into a high-level executive summary with professional visualizations.

### Key Features:
- **Pandas Integration:** Uses vectorized operations for statistical calculations (`mean`, `sum`, `max`).
- **Data Categorization:** Employs `groupby()` and `value_counts()` to identify spending patterns.
- **Visualization:** Generates high-quality pie charts using `Matplotlib` to provide a visual breakdown of expenses.
